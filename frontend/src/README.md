# Next.js app source

- **app/** — App Router routes: landing, map page, dashboard, etc.
- **components/** — Map, POI cards, payout/yield displays, wallet connect.
- **hooks/** — useContract, useWallet, useChainState, etc.

Use React + Next.js; for maps consider Mapbox GL JS or Leaflet (no Google). Wallet: Wagmi + viem, or RainbowKit, or Flare-recommended wallet libs.
